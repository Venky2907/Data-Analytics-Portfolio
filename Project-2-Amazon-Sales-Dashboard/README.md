# 📊 Amazon Sales Dashboard

## 🔹 Problem Statement
Analyze e-commerce sales data to identify key trends in sales, orders, and customer behavior.

---

## 🔹 Solution
Developed an interactive Power BI dashboard to track sales performance across cities, categories, and order status.

---

## 🔹 Tools Used
- Power BI (Visualization, DAX)
- Excel / CSV (Data Source)

---

## 🔹 Key Features
- Sales overview (Total Sales, Orders, Quantity, AOV)
- Top 10 cities by sales
- Sales by category and state
- Order status distribution (Shipped, Cancelled, Returned)
- Sales by fulfillment (Amazon vs Merchant)

---

## 🔹 Key Insights
- Bengaluru, Hyderabad, and Mumbai are top-performing cities
- Majority of orders are successfully shipped
- Amazon fulfillment contributes higher sales than merchant
