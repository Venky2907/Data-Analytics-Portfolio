# 📊 Plant KPI Dashboard (Operations Excellence)

## 🔹 Problem Statement
In manufacturing operations, multiple departments (Safety, Quality, SCM, Maintenance, Production) track KPIs separately, leading to lack of centralized visibility and delayed decision-making.

---

## 🔹 Solution
Developed a centralized Power BI dashboard to track plant-level KPIs across all departments for Tier-3 management meetings.

---

## 🔹 Tools & Technologies
- Power BI (DAX, Data Modeling, Visualization)
- Power Query (ETL Automation)
- SQL (Data Extraction & Validation)
- Excel, SharePoint (Data Sources)
- IoT Device Data (Machine KPIs)

---

## 🔹 Key Features
- Integrated multiple data sources (Excel, SharePoint, IoT devices)
- Designed star schema data model with relationships
- Built advanced DAX measures (KPI tracking, trends, variance)
- Automated data refresh using Power BI Gateway
- Enabled real-time KPI monitoring before Tier-3 meetings

---

## 🔹 Business Impact
- Eliminated manual reporting efforts
- Improved decision-making speed for leadership
- Provided single source of truth for all departments
- Automated KPI tracking even in absence of manual intervention
